GRGOV Mobile Search

Γενικά

Η εφαρμογή δίνει τη δυνατότητα αναζήτησης πλήρους κειμένου σε πολύ σημαντικές πηγές ανοικτών δεδομένων της Ελλάδος από κινητά τηλέφωνα (smartphones) και tablets που χρησιμοποιούν λειτουργικό σύστημα Android και iOS.

Πηγές δεδομένων

  1. Όλα τα έγγραφα που αναρτώνται στο πρόγραμμα Δι@ύγεια diavgeia.gov.gr. Η ανάκτηση γίνεται από την ΥπερΔιαύγεια μέσω του API ανοικτών δεδομένων του προγράμματος Διαύγεια http://opendata.diavgeia.gov.gr/.
  2. Το σύνολο της Ελληνικής Νομοθεσίας που δημοσιεύεται από Εθνικό Τυπογραφείο http://www.et.gr/. Η ανάκτηση των Φύλλων Εφημερίδας της Κυβερνήσεως (ΦΕΚ) γίνεται από την ΥπερΔιαύγεια με web scraping σε συνεργασία με τους Μιχάλη Μοάτσο και Στέφανο Καλαντζή.
  3. Όλες τις προκηρύξεις και τις προμήθειες που δημοσιεύονται στο Κεντρικό Ηλεκτρονικό Μητρώο Δημοσίων Συμβάσεων (ΚΗΜΔΗΣ) http://www.eprocurement.gov.gr/. Η ανάκτηση γίνεται από την ΥπερΔιαύγεια και το εργαλείο DEiXTo με web scraping. Για περισσότερες πληροφορίες: Building an RSS feed for the Greek e-procurement platform

Τεχνολογία

Για την υλοποίηση της εφαρμογής χρησιμοποιούνται οι εξής τεχνολογίες:

  • Python
  • Nginx
  • HTML5
  • OpenSearch API
  • Javascript
  • Ratchet 2.0
  • ΥπερΔιαύγεια: H εφαρμογή χρησιμοποιεί τα ανοικτά δημόσια δεδομένα κατόπιν επεξεργασίας από την ΥπερΔιαύγεια.

Κώδικας

Ο πλήρης κώδικας της εφαρμογής διατίθεται ελεύθερα σε κάθε ενδιαφερόμενο από το github https://github.com/vbanos/grgov-mobile-search.
